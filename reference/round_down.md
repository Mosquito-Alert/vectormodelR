# Round a given number x downward to the nearest n.

Round a given number x downward to the nearest n.

## Usage

``` r
round_down(x, n)
```

## Arguments

- x:

  The number to be rounded.

- n:

  The number to be rounded.

## Value

The numeric result of the downward rounding.

## Examples

``` r
round_down(4.569, 0.05)
#> [1] 4.55
```
