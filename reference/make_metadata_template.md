# Make metadata template ready for data portal

Make metadata template ready for data portal

## Usage

``` r
make_metadata_template(
  type = "Dataset",
  name = "",
  description = "",
  conditions_of_access = "",
  url = "",
  same_as = "",
  identifier = "",
  license = "",
  citation = "",
  measurement_technique = "",
  temporal_coverage = "",
  creator = "",
  variable_info = "",
  distribution = ""
)
```

## Arguments

- type:

  A string.

- name:

  A string.

- description:

  A string

- conditions_of_access:

  A string

- url:

  A string

- same_as:

  A string

- identifier:

  A string

- license:

  A string

- citation:

  A string

- measurement_technique:

  A string

- temporal_coverage:

  A string

- creator:

  A string

- variable_info:

  A string

- distribution:

  A string

## Value

A json string.
